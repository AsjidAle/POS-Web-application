# POS-Web-application
A general prototype Point of sale Web Application for small business
<h2>Users</h2>
<ul>
  <li>Admin-Owner</li>
  <ul>
    <li>Add new Users</li>
    <li>Remove Users</li>
    <li>Analyze sales</li>
    <li>Analyze inventory</li>
    <li>Analyze top selling product</li>
    <li>Getting notice of ending products</li>
  </ul>
  <li>Sales staff</li>
  <ul>
    <li>Search Items</li>
    <li>Record sales</li>
  </ul>
  <li>Return staff</li>
  <ul>
    <li>Search Items</li>
    <li>Return Items</li>
  </ul>
  <li>Management staff</li>
  <ul>
    <li>Add new items</li>
    <li>Update items(just add more)</li>
  </ul>
</ul>
